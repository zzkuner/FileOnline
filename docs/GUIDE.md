# InsightLink 使用指南

## 🎯 快速体验流程

### 第一步：访问应用
打开浏览器访问：http://localhost:3000

您会看到精美的 Landing Page，包含：
- 渐变色的 Hero 区域
- 功能特性展示
- 应用场景说明
- CTA 按钮

### 第二步：注册账户

1. 点击右上角"开始使用"或"注册"按钮
2. 填写注册信息：
   - 姓名：张三
   - 邮箱：test@example.com
   - 密码：123456（至少6位）
   - 确认密码：123456
3. 点击"注册账户"
4. 注册成功后自动跳转到登录页

### 第三步：登录系统

1. 使用刚才注册的邮箱和密码登录
2. 登录成功后进入仪表盘

### 第四步：上传文件

在仪表盘页面：
1. 看到大大的上传区域（带虚线边框）
2. 点击"选择文件"按钮
3. 选择一个文件（建议先测试 PDF 或图片）
4. 上传成功后，文件会出现在下方的"我的文件"列表中

### 第五步：创建追踪链接

1. 点击刚上传的文件卡片
2. 进入文件详情页，看到：
   - 文件统计信息（链接数、访问量、创建时间）
   - "创建追踪链接"按钮
3. 点击"创建追踪链接"
4. 填写链接信息：
   - 链接名称：投递腾讯（必填）
   - 描述：2025春招简历投递（可选）
5. 点击"创建链接"
6. 创建成功后，链接会出现在下方列表中

### 第六步：分享链接

在链接列表中：
1. 看到刚创建的链接卡片
2. 链接格式：`http://localhost:3000/v/{随机字符串}`
3. 点击"复制"图标复制链接
4. 点击"打开"图标在新标签页中预览

### 第七步：访客查看（模拟）

1. 打开新的无痕窗口（模拟访客）
2. 粘贴刚才复制的链接
3. 访客会看到：
   - 干净的文件查看器界面
   - 文件内容展示
   - 底部"由 InsightLink 安全分享"提示
4. 后台会自动记录：
   - 访问时间
   - 设备信息
   - 浏览器类型
   - 停留时长

### 第八步：查看分析数据

回到原窗口：
1. 在文件详情页，找到刚才创建的链接
2. 点击"查看分析"按钮
3. 进入分析页面，可以看到：
   - **统计卡片**：总访问量、桌面访问、移动访问、平均时长
   - **访问记录列表**：每个访客的详细信息
     - 访客编号
     - 设备类型（Desktop/Mobile）
     - 访问时间
     - 停留时长
     - 浏览器和操作系统
     - IP 地址
     - 事件记录（PAGE_VIEW、HEARTBEAT 等）

## 🔥 核心追踪功能

### 自动追踪的数据

1. **访问信息**
   - 访问时间（精确到秒）
   - 访问时长（自动计算）
   - 访问次数

2. **设备信息**
   - 设备类型：Desktop/Mobile/Tablet
   - 浏览器：Chrome/Firefox/Safari 等
   - 操作系统：Windows/macOS/iOS/Android

3. **网络信息**
   - 访客 IP 地址
   - 来源页面（Referrer）

4. **行为事件**
   - PAGE_VIEW：页面查看
   - HEARTBEAT：心跳（每30秒）
   - SESSION_END：会话结束
   - VIDEO_PROGRESS：视频播放进度（视频文件）
   - PDF_PAGE_VIEW：PDF 翻页（待实现）

### 追踪机制

1. **初始化追踪**
   - 访客打开链接时，自动发送 PAGE_VIEW 事件
   - 创建 Visit 记录

2. **心跳追踪**
   - 每30秒发送一次 HEARTBEAT 事件
   - 用于计算准确的停留时长

3. **会话结束**
   - 访客关闭页面时发送 SESSION_END 事件
   - 更新访问时长

## 📊 数据流程

```
用户上传文件
    ↓
创建追踪链接（生成唯一 slug）
    ↓
分享链接给访客
    ↓
访客打开链接
    ↓
自动记录访问信息（Visit）
    ↓
持续记录事件（Event）
    ↓
用户查看分析数据
```

## 🛠️ 测试建议

### 测试场景 1：基础流程
1. 注册 → 登录 → 上传文件 → 创建链接 → 查看分析
2. 验证每个步骤是否正常工作

1. 为同一个文件创建多个链接
2. 分别打开不同的链接
3. 验证数据是否正确分离

### 测试场景 3：设备识别
1. 用桌面浏览器打开链接
2. 用手机浏览器打开链接
3. 查看分析页面，验证设备类型识别

### 测试场景 4：时长统计
1. 打开链接后停留一段时间（如2分钟）
2. 关闭页面
3. 查看分析，验证时长是否准确

## 🐛 常见问题

### Q: 上传文件失败？
A: 检查文件大小是否超过限制（当前支持最大 10MB）

### Q: 链接打开显示"链接不存在"？
A: 检查链接是否完整复制，或链接是否被禁用

### Q: 访问数据不准确？
A: 确保访客完整加载了页面，并且没有被广告拦截器阻止

### Q: 视频无法播放？
A: 确保视频格式为 MP4，浏览器支持 H.264 编码
