## 🚀 项目特色

### 1. 一文多链
同一个文件可以创建无限个追踪链接，精准识别访客来源。

### 2. 实时追踪
访客打开链接后自动开始追踪，无需任何操作。

### 3. 详细分析
提供访客设备、浏览器、访问时长等详细信息。

### 4. 精美 UI
现代化的设计，玻璃态效果，动画流畅。

### 5. 隐私保护
文件查看器禁止下载，保护源文件安全。
